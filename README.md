# colorization-practice
Python-скрипт, реализующий подход из научной работы Gal et al., "Colorization using Optimization" (SIGGRAPH 2015). Скрипт использует оптимизацию в цветовом пространстве L*a*b*, суперпиксельную сегментацию и разреженные линейные системы для восстановления цветных каналов при сохранении оригинальной яркости.
